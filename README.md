# textlint-rule-preset-weseek

textlint rule preset 

## Installation

### npm
```bash
npm install textlint-rule-preset-weseek
```

### yarn
```bash
yarn textlint-rule-preset-weseek
```

## Usage

Via `.textlintrc`

```bash
{
    "rules": {
        "preset-weseek": true
    }
}
```
